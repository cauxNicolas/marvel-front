# Marvel

> Realization of a Marvel website with React. We had to connect to the Marvel Developpeur database, retrieve the data and display it.

## Site composition

- Login
- Register
- Personnages
- Comics
- Favoris

## Technologies

- axios - versions 0.19.2
- React - versions 16.13.1
